# Install Nothing — Terminal Simulator

A playful, realistic terminal you can run in the browser. Choose a theme, pick a tech‑stack preset, set a duration or go Endless, then watch a convincing installer simulation — powered by our CLI and streamed live to your screen.

## Use Cases
- Conference and booth demos without a fragile live setup
- Product teasers and landing pages with motion that feels real
- Stream overlays/backgrounds for developer content
- Classroom exercises and onboarding tours
- Portfolio pages to showcase terminal UIs
